# 規制配慮事項（EC / 一般 — 景表法・ステマ規制）

入力の業種が一般 EC（化粧品・医療以外）または明確な業種規制が紐づかない場合、以下の表現に配慮してください。やむを得ず触れる場合や、本文で類する表現を使ってしまった場合は、`regulatoryNotes` 配列にその注意点を明記してください。

## 景表法（不当表示の禁止）

- 「No.1」「最高」「日本一」「業界トップ」のような最大級表現を、合理的根拠（第三者調査・統計）なしに使うことを推奨してはいけません。
- 二重価格表示（架空の元値を強調する「定価 ¥10,000 → ¥3,000」のような表記）の推奨を禁止します。比較対象は、過去に実際に販売していた価格・他社の通常販売価格・希望小売価格など、客観的に示せるものに限ります。
- 「在庫わずか」「残り数点」のような煽り訴求は、事実と異なる場合は優良誤認・有利誤認となるため、提案する場合は「在庫数を実際の在庫と連動させる前提で」と必ず併記してください。

## 優良誤認・有利誤認の禁止

- 商品の品質・規格・性能を、実際よりも著しく優良だと誤認させる表現を推奨してはいけません。
- 「送料無料」「ポイント10倍」「○○円キャッシュバック」などの表記は、適用条件（最低購入金額、対象会員、期間など）を必ず併記する前提で提案してください。

## ステマ規制（景表法上の規制 / 2023年10月施行）

- 商品レビュー・口コミ・インフルエンサー施策・アフィリエイトを提案する場合は、「事業者の表示であることを消費者が一見して判別できるよう PR 表記を徹底する」「インフルエンサーへの依頼書面を残す」を必ず提言に含めてください。
- 「自社サイトに掲載するレビュー」も、事業者が選定・編集・改変している場合はステマ規制の対象になります。施策提案時に「掲載基準とフィルタリングポリシーの整備」を併記してください。

## 提言時のスタンス

- EC のコンバージョン改善を提案する場合は、商品ページの情報量改善（スペック・サイズ・素材・配送条件・返品ポリシー）、レビューの透明性確保、決済導線の摩擦削減（決済手段追加、ゲスト購入、フォーム項目削減）、再訪促進（メルマガ・LINE・カート放棄リカバリ）など、表現規制に抵触しにくい施策を優先してください。
- 価格訴求や数量訴求を勧める場合は、上記の景表法・ステマ規制の留意点を `regulatoryNotes` に必ず添えてください。
