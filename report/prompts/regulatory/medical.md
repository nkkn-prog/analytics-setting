# 規制配慮事項（医療 / 医療広告ガイドライン）

入力の業種が医療の場合、提言や本文で**以下の表現を推奨してはいけません**。やむを得ず触れる場合や、本文で類する表現を使ってしまった場合は、`regulatoryNotes` 配列にその注意点を明記してください。

## 治療効果の断定の禁止

- 「必ず治る」「100%改善」「絶対安全」「副作用がない」のように、治療効果や安全性を断定する表現を提案してはいけません。
- 「最先端の治療」「日本でも数少ない」のような優越性を示唆する表現も、客観的根拠がない限り使えません。

## ビフォーアフター訴求の禁止（限定的許可）

- 治療前後の画像（ビフォーアフター）を煽る形で訴求するページや LP の作成を推奨してはいけません。
- 厚労省ガイドラインでは、術前・術後の写真は治療内容・費用・主なリスク・副作用等の詳細な説明を併記する場合のみ許容されます。安易な「効果を見せる」訴求の推奨はしないでください。

## 体験談を CV 訴求の中心に据えることの禁止

- 患者の体験談・レビューを集めて訴求する施策を主軸に据える提案はしないでください。医療広告では患者個人の体験談は規制対象です。
- 既存の口コミ機能を改善する提案を行う場合は、「掲載前にスタッフが医療広告ガイドラインに照らして審査するフロー」を必ず併記してください。

## 比較優良広告の禁止

- 「日本一」「No.1」「最高水準」「他院より優れた」のような比較優良広告を推奨しないでください。客観的事実に基づく場合でも、医療広告ではほぼ使用不可と認識すべきです。

## 提言時のスタンス

- 医療機関の予約コンバージョン改善を提案する場合は、客観的事実の整理（診療科目・診療時間・アクセス・医師経歴・対応疾患・予約方法）を分かりやすく見せる UI 改善や、地域名 + 診療科の検索流入を取りこぼさないコンテンツ拡充を中心に提案してください。
- 限定的な訴求（料金、初診料、駐車場、バリアフリー、Web 予約の利便性など）は問題ない範囲ですので、こちらを優先して提言してください。
